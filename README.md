# Python Expense Tracker

A command-line expense tracking application built with Python. The program allows users to record, view, summarize, and delete expenses while saving data locally in a JSON file.

## Features

* Add expenses with a description, category, amount, and date
* View all recorded expenses
* Calculate the total amount spent
* View spending totals by category
* Delete expenses
* Save expenses permanently using JSON
* Validate user input and handle common errors

## Technologies Used

* GitHub

## How to Run

1. Clone the repository:

```bash
git clone YOUR_REPOSITORY_URL
```

2. Open the project folder:

```bash
cd expense-tracker
```

3. Run the program:

```bash
python3 main.py
```

## Example

```text
Expense Tracker

4. Delete expense
5. View totals by category
6. Exit
```
